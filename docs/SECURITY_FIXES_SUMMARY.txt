HOW TO USE
================================================================================

LOCAL DEVELOPMENT:
1. Copy .env.dev.example to .env
2. Update values if needed
3. Run: mvn spring-boot:run -Dspring-boot.run.profiles=dev

KUBERNETES (STAGING/PROD):
1. Create namespace: kubectl create namespace job-portal
2. Create secrets: kubectl create secret generic jobportal-secrets ...
3. Update ConfigMap: Edit k8s/jobportal-config.yaml
4. Deploy: kubectl apply -f k8s/ -n job-portal
